
# uncertainty calculator

A class-based Flask backend using SymPy for symbolic evaluation and uncertainty propagation with support for any number of variables.

## Features
- Dynamic symbolic expressions
- Automatic uncertainty propagation
- Any number of variables
- Fully class-based architecture

## Installation
```bash
git clone https://github.com/yourname/flask-sympy-backend.git
cd flask-sympy-backend
python -m venv venv
source venv/bin/activate
